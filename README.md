# Bank Login System – Java GUI

A basic login system built using Java Swing with a background image.

## 🧩 Features
- User login form with username & password
- Background image setup using JLabel
- Basic validation
- Swing layout design

## 🛠️ Technologies Used
- Java
- Java Swing

## 🧠 What I Learned
- GUI layout design
- Event handling
- Using images in Java apps
